# 🚀 Features

ShadowTrace provides multiple modules for network traffic analysis.

---

## Dashboard
- Visualizes attack distribution
- Shows high-traffic patterns
- Displays key metrics (flows, attacks)

---

## NLP Query System
- Converts natural language to SQL
- Allows users to ask questions like:
  - "Show most common attacks"
  - "Find high traffic flows"

---

## Real-Time Simulation
- Generates live traffic data
- Detects high packet rate anomalies
- Enables dynamic analysis

---

## Learning Module
- Explains cyber attacks in simple terms
- Provides risk level and prevention methods

---

## Hybrid Data Mode
- Switch between:
  - Database (historical data)
  - Simulation (real-time data)

---

## Intelligent Insights
- Identifies dominant attacks
- Highlights high-risk traffic patterns

---

##  Goal
To provide an interactive system for:
- Understanding network traffic
- Detecting anomalies
- Learning cybersecurity concepts
